# End-to-End ETL Pipeline using Databricks & PySpark

## Overview
Built an end-to-end ETL pipeline using Databricks, PySpark, and Delta Lake following Medallion Architecture.

## Architecture
Bronze → Silver → Gold

## Tech Stack
- Databricks
- PySpark
- Delta Lake
- SQL
- AWS S3

## Features
- Incremental loading
- Data transformation
- Delta tables
- CDC handling
- Data quality validation

## Workflow
1. Ingest raw data
2. Load into Bronze layer
3. Clean and transform in Silver layer
4. Aggregate into Gold layer
